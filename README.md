# README

TO RUN:
* npm install
* bundle install
* bundle exec rails db:drop db:setup

* rails server
* webpack --watch

* Navigate to http://localhost:3000

PUSHING TO HEROKU:
* git push heroku master

* (seeding heroku database)
* heroku run rake db:migrate (if pending migrations)
* heroku run rake db:seed
